# High & Low Game (PHP版)

ブラウザで遊べるハイアンドローゲームです。
PHP7以上とブラウザで動作します。

---

## ゲーム内容
- ランダムに生成されたカードの数値を予想
- 「次のカードは今のカードより高いか低いか」を選択
- 予想が当たればゲーム続行可能、外れればゲーム終了
- ブラウザ上で直感的に操作可能

## 工夫した点
- **ランダム生成ロジック**：公平にカードが選ばれるよう設計
- **セッション管理**：ゲーム状態（現在のカード・スコア）を維持
- **UI工夫**：シンプルかつ見やすいデザイン
- **入力チェック**：誤操作や不正入力への対応
